Como funcionam as âncoras?

(Voltar)

As âncoras nada mais são do que tags que permitem o uso de link em textos, são os chamados hyperlinks.

Veja a seguir um exemplo de como usá-los: <a href="endereço-do-link">Texto</a>

O que são Clients?

(Voltar)

Por definição, Clients são os computadores que acessam a internet e que têm acesso aos sites, aplicativos e programas, que são armazenados pelos servidores.

Quais são os atributos básicos de uma tag?

(Voltar)

Uma tag sempre terá alguns atributos que é comum para todas as tags, como por exemplo os atributos name, id, class e etc. Também existem os atributos únicos que são específicos de cada tag, como por exemplo o width, que é utilizado na tag <img>

Como é uma estrutura básica de um HTML?

(Voltar)

No HTML, temos o cabeçalho, que é onde colocamos todas as coisas que definem como a página vai se comportar, como por exemplo a responsividade, e também temos o corpo do texto, que é onde a página toma forma

Não esqueça também de colocar o link para voltar até o índice. (Lembra das âncoras?)

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas, como o próprio nome já diz, ordena os itens quando colocamos os itens dentro dela, com 1, 2, 3 e por aí vai, já na lista não ordenada isso não acontece, e isso é útil quando só queremos fazer uma lista qualquer, com o nome de cidades por exemplo.
